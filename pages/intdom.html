Interactuando con el DOM

Los elementos en el DOM se organizan jerárquicamente según se han creado las diferentes etiquetas en el documento html.

Para interaccionar con los elementos del DOM se utilizan diferentes métodos. Los más habituales son:


  • getElementById(), por ID
  • querySelector(), por selector
  • querySelectorAll(), selector de varias coincidencias
  • document.getElementsByTagName(), por etiqueta
  • document.getElementsByClassName(), por clase


Ejemplo de consola

 
    document.getElementById("tituid").innerHTML = "Introducción a DOM";
    document.querySelector(".parraqs").innerHTML = "Párrafo con queryselector";

    var listas = document.querySelectorAll(".lisqs");
    // console.log(listas);
    for (let i = 0; i < listas.length; i++) {
        listas[i].style.color = "red";
    }

    var parrafoss = document.getElementsByTagName("p");
    console.log(parrafoss);

    var listas2 = document.getElementsByClassName("lisqs");
    console.log(listas2);

Este párrafo se cambiará por querySelector

  • querySelector(), por selector
  • querySelectorAll(), selector de varias coincidencias


Manipulación del DOM y creación de elementos
Se puede definir donde se quiere insertar el nuevo código HTML con insertAdjacentHTML(). El primer parámetro marca justamente la posición donde se insertará:

  • beforebegin, para insertar antes del elemento
  • afterbegin, para insertar despues del elemento
  • beforeend, para insertar antes del elemento
  • afterend, para insertar despues del elemento

También podemos crear elementos con createElement() y luego anadirlos con appendChild(). Si queremos eliminarlos podemos utilizar remove() 


 
            
    var articulo = document.createElement("p");
    articulo.innerHTML = "Nuevo parrafo";
    document.getElementById("articulo").appendChild(articulo);

    document.getElementById("borrar").remove();

    var parseo = document.getElementById("parsear");
    parseo.insertAdjacentHTML(
        "beforebegin",
        "<span class='nueva'>Esto es un texto</span>"
    );
