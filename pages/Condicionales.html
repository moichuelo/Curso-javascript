Condicionales

Para comprobar si se cumple una condición se utilizan los condicionales if/else y switch

IF / ELSE / ELSE IF

El condicional If permite ejecutar un bloque de código si se cumple una condición
El condicional Else permite ejecutar un bloque de código si no se cumple la condición
El condicional Else If permite evaluar múltiples condiciones

 
                if (edad >= 18) {
                    texto = "Eres mayor de edad";
                    if (nombre == "Javier") { //if anidado
                        texto = texto + " y te llamas Javier";
                    } //fin del if
                    document.getElementById("parrafov").innerHTML = texto;
                } else {
                    document.getElementById("parrafov").innerHTML = "Eres menor de edad";
                }


                let date = new Date(); //función nativa de Js que devuelve la fecha y hora actuales
                let hora = date.getHours(); //función nativa de Js que extrae la hora de la variable date
            
                if (hora >= 6 && hora < 12) {
                    document.getElementById("parrafov2").innerHTML = "Buenos dias";
                } else if (hora >= 12 && hora < 20) {
                    document.getElementById("parrafov2").innerHTML = "Buenas tardes";
                } else {
                    document.getElementById("parrafov2").innerHTML = "Buenas noches";
                }


                let hora = date.getHours();
            
                if (hora >= 6 && hora < 12) { //primera condición
                    document.getElementById("parrafov2").innerHTML = "Buenos dias";
                } else if (hora >= 12 && hora < 20) { //comprobación de una segunda condición
                    document.getElementById("parrafov2").innerHTML = "Buenas tardes";
                } else { //en caso de no cumplirse ninguna de las dos condiciones
                    document.getElementById("parrafov2").innerHTML = "Buenas noches";
                }

SWITCH

El condicional Switch se utiliza para evaluar múltiples condiciones 

 
                Var date = new Date();
                Var día = date.getDay(); //función nativa de Js que extrae el día de la semana (guarda un valor de 0 a 6, siendo 0 el domingo)
                switch (día) {
                case 0:
                    day = "Domingo";
                    break;
                case 1:
                    day = "Lunes";
                    break;
                case 2:
                    day = "Martes";
                    break;
                case 3:
                    day = "Miércoles";
                    break;
                case 4:
                    day = "Jueves";
                    break;
                case 5:
                    day = "Viernes";
                    break;
                case 6:
                    day = "Sábado";
                } 

                console.log(day);


                switch (día) { //se pueden agrupar varias condiciones
                    case 0:
                    case 6:
                        day = "Es fin de semana";
                        break;
                    case 1:
                    case 2:
                    case 3:
                    case 4:
                    case 5:
                        day = "Toca currar";
                }

                switch (nombre) { //si quieremos que se ejecute un código en caso de no haberse cumplido ninguna condición podemos utilizar default
                    case “Juan”:
                        console.log(“Hola Juan”)
                        break;
                    case “Pedro”:
                        console.log(“Hola Pedro”)
                        break;
                    default:
                        day = "A ti no te conozco";
                }
