Bucles

Los bucles pueden ejecutar un bloque de código varias veces

For

El bucle for se utiliza cuando sabemos el número exacto de veces que se tiene que repetir un bloque.

El bucle for declara 3 expresiones separadas por ;

  • La primera es la variable de control (se ejecuta antes de iniciar el bucle)
  • La segunda es la condición que se evalua en cada iteración
  • La tercera es el incremento que tenemos en cada iteración
 
    for (let i = 0; i < 10; i++) {
        document.getElementById("bucles").innerHTML += i + " ";
    }

    for (let i = 1; i <= 6; i++) {
        document.getElementById("bucles").innerHTML +=
            "<h" + i + ">Titulo " + i + "</h" + i + ">";
    }

While

El bucle while declara 1 expresión, la condición que se evalua en cada iteración. No necesitamos saber el número de repeticiones

 
    let pregunta = window.prompt("Dime tu nombre");

    let i = 1;
    while (pregunta != "Juan") {
        pregunta = window.prompt("Dime tu nombre");
        i++;
        if (i == 3) {
            document.getElementById("bucles2").innerHTML =
                "Has fallado 3 veces";
            break;
        }
    }

    if (i != 3) {
        document.getElementById("bucles2").innerHTML = "Hola Juan";
    }

Do while

Variante del bucle while que se ejecuta al menos una vez ya que evalúa la condición tras ejecutar el bloque 

 
                do {
                    pregunta = window.prompt("Dime tu nombre");
                } while (pregunta != "Juan");
For in

Recorre las propiedades de un objeto o un array

 
    const person = {fname:"John", lname:"Doe", age:25};

    let text = "";
    for (let x in person) {
    text += person[x];
    } //John Doe 25

    const numbers = [45, 4, 9, 16, 25];

    let txt = "";
    for (let x in numbers) {
    txt += numbers[x];
    } //45 4 9 16 25
For each

El método forEach() llama a una función por cada elemento de un array.
La función recibe 3 valores:

  • value: el valor del elemento actual
  • index: la posición del elemento actual
  • array: el array completo
 
    const numbers = [45, 4, 9, 16, 25];

    let txt = "";
    numbers.forEach(myFunction);
    document.getElementById("demo").innerHTML = txt;

For of

Recorre los valores de un objeto iterable

 
    const cars = ["BMW", "Volvo", "Mini"];

    let text = "";
    for (let x of cars) {
        text += x;
    } //BMW Volvo Mini

    let language = "JavaScript";

    let text = "";
    for (let x of language) {
    text += x;
    } //J a v a S c r i p t
