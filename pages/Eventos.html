Eventos y escuchadores

Los eventos son aquello que sucede mientras el usuario interactua con el navegador. Los eventos más comunes son:


  • click, para detectar cuando se hace click
  • scroll, para detectar cuando se hace scroll
  • submit, para detectar cuando se envia un formulario
  • change, para detectar cuando cambia el contenido de un elemento
  • focus, para detectar cuando entra el foco del elemento
  • blur, para detectar cuando sale el foco del elemento
  • mouseover, para detectar cuando entra el ratón en el elemento
  • mouseout, para detectar cuando sale el ratón del elemento
  • mousemove, para detectar el movimiento del ratón sobre el elemento
  • keydown, para detectar la tecla pulsada
  • keyup, para detectar la tecla soltada
  • load, para detectar la carga de la página o un elemento
  • resize, para detectar el cambio de tamaño de la ventana


Para controlar los eventos se utiliza addEventListener()

 
    document.querySelector("#formulario")
        .addEventListener("submit", function(evento) {
        console.log('Se ha enviado el formulario');
    });
